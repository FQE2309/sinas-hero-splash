# Carpeta de Videos - Tutoriales SINAS

Esta carpeta contiene los videos tutoriales disponibles para reproducción en la landing page del SINAS.

## Formatos de video recomendados:

### Formatos soportados:
- **MP4** (H.264/AVC) - Recomendado
- **WebM** (VP9) - Para navegadores modernos
- **OGV** (Theora) - Compatibilidad adicional

### Estructura sugerida:
```
videos/
├── ingreso-sistema.mp4
├── navegacion-sistema.mp4
├── cargue-alcaldias.mp4
├── inventario-rural.mp4
├── gestion-iniciativas.mp4
└── README.md
```

## Mejores prácticas:

### Optimización:
- **Resolución:** 1280x720 (HD) o 1920x1080 (Full HD)
- **Bitrate:** 2-5 Mbps para HD, 5-8 Mbps para Full HD
- **Duración:** Máximo 15-20 minutos por video
- **Compresión:** H.264 con AAC audio

### Naming convention:
- Usar kebab-case (guiones)
- Nombres descriptivos sin espacios
- Evitar caracteres especiales

### Metadatos:
- Incluir título en español
- Descripción breve del contenido
- Duración aproximada

## Herramientas de conversión recomendadas:
- **FFmpeg** - Para conversión por línea de comandos
- **Handbrake** - Interfaz gráfica
- **Adobe Media Encoder** - Para usuarios avanzados
